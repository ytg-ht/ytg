How to use (important)

  1. Click Generate 7 Facts.
  2. Click Open Preview Popup (a 1080×1920 popup opens).
  3. Click Record Popup & Download. When the browser asks what to share, choose the PREVIEW POPUP WINDOW (this ensures music + voice are captured).
  4. Wait — the popup plays 7 facts, then the download will start automatically (file: short.webm).

If popup is blocked: allow popups for this site. If voices don't appear, refresh both pages and re-open popup (Chrome sometimes loads voices late).
